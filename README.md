 Pharmacy Management System
A full-stack Flask-based Pharmacy Management System designed to streamline and automate core pharmacy operations such as medicine inventory, sales tracking, customer management, supplier management, and purchase records. The system provides a centralized dashboard for real-time insights and efficient business control.

 Project Overview:
This application helps pharmacy owners manage day-to-day operations digitally, replacing manual record-keeping with a secure, structured, and scalable web solution. It includes authentication, role-based access (session-based login), and a clean modular architecture using Flask Blueprints.
The system ensures accurate inventory tracking, prevents stock shortages, and maintains detailed transaction history for better decision-making.

Key Features:
User Authentication (Login, Register, Logout with session handling)
Medicine Inventory Management (Add, Edit, Delete, Update Stock)
Sales Management with billing records
Purchase tracking from suppliers
Customer and Supplier management modules
Dashboard with real-time statistics including total medicines, total sales, low stock, and expired medicines
Total medicines
Total sales
Low stock alerts
Expired medicines tracking
Secure session-based access control
 Modular Flask architecture (Blueprint-based structure)
 
 Tech Stack:
Backend: Python
Database:  MySQL
ORM: SQLAlchemy
Frontend: HTML, CSS, JavaScript
Authentication: Flask Sessions + Werkzeug Security
Deployment: Render / Gunicorn
